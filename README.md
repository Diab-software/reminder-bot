# Reminder Bot

بوت Discord بسيط لتسجيل التذكيرات.

## المتطلبات
- Python 3.8+
- مكتبة discord.py

## طريقة التشغيل على Replit
1. افتح [Replit.com](https://replit.com) وسجّل دخولك.
2. أنشئ Repl جديد → اختر **Python**.
3. ارفع الملفات الثلاثة (`reminder_bot.py`, `requirements.txt`, `README.md`) في مشروعك.
4. افتح [Discord Developer Portal](https://discord.com/developers/applications) وأنشئ بوت، انسخ التوكن.
5. ضع التوكن في `reminder_bot.py` مكان `"TOKEN_BOT_HERE"`.
